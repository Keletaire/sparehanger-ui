  • ¶

    A row defines a single horizontal section of the site.

    .ui-row {
	display: block;
	margin: 30px 0;
  • ¶

    It uses a number of tricks to ensure that rows are always placed ontop of one another.

    	&:before,
	&:after {
		display: table;
		content: " ";
	}

	&:after {
		clear: both;
	}
}
  • ¶

    A column is a verical section of a row. Its width - specified in the class itself - is actually applied to it using javascript.

    [class*='ui-column-'] {
	display: inline-block;
	float: left;
  • ¶

    When the screen size is phone or small-tablet sized, all columns stack.

    	@media screen and (max-width: 640px){
		display: block;
		float: none;
		width: 100% !important;
		margin: 10px 0 !important;
	}
}
