    This can get very tricky. We want the colors of the dropdown options to invert when we hover over them. However, the link styling of whatever the dropdown is used in can override this. Therefore, we apply the !important flag to make sure it is chosen.

    		&:hover {
			background: $dark-blue;
			color: #FFF !important;
		}

		&:first-child {
			border-top: 0;
		}

		a {
			display: block;
			padding: 3px 20px;
			color: $dark-blue;

			&:hover {
				color: #FFF !important;
			}
		}
	}
