Spare Hanger Style Guide

Grid

Spare Hanger uses a grid system based on percentages. For instance, a block-level element such as 

<div class="column-100">

would cover 100% of the width of its parent element. Typical usage of the grid system might look like this: 


<div class="row">
	<div class="column-30">
		This is a column that is 30% of the width
	</div>
	<div class="column-70">
		This is a larger column this is 70% of the width
	</div>
</div>

Note that

<div class="row">

is required and that the percentages in a .row must add up to 100%.
